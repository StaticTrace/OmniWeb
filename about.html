About

OmniWeb is a minimal personal site template with a modular dashboard for live widgets.

It's designed to be lightweight, fast, and easy to customize without the overhead of heavy frameworks.

Why OmniWeb?

  • No Dependencies: Pure HTML, CSS, and JavaScript
  • Dark Mode: Built-in dark/light theme support
  • Responsive: Mobile-first design that works everywhere
  • Modular: Easy to add or remove components
  • Fast: Minimal file sizes for quick loading
