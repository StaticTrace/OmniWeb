# OmniWeb

A lightweight, zero-dependency personal website and dashboard template with live widgets.

## Features
- ✨ **No Dependencies** - Pure HTML, CSS, Vanilla JS
- 🌙 **Dark Mode** with persistence
- 📱 **Fully Responsive**
- ⚡ **Extremely Fast**
- 🛠 **Tools Page** with 5 utilities

## Project Structure (Flat Root)
All files are in the root directory as requested.

## Getting Started
1. Clone: `git clone https://github.com/StaticTrace/OmniWeb.git`
2. Serve: `python -m http.server 8000`
3. Open: http://localhost:8000

Live demo (GitHub Pages): https://StaticTrace.github.io/OmniWeb

## Tools
- Text Counter
- Secure Password Generator
- Random Name Generator
- Calculator
- JSON Formatter
